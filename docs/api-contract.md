# Contrato da API

## Health

- `GET /health`

## Usuarios

- `POST /api/users/register`
- `GET /api/users`
- `GET /api/users/:id`
- `PUT /api/users/:id`
- `DELETE /api/users/:id`
- `DELETE /api/users`

## Autenticacao

- `POST /api/auth/login`
- `GET /api/auth/me`

## Senha

- `POST /api/password/forgot`
- `POST /api/password/reset`

## Formularios

- `POST /api/forms/contact`
